# Amazon Product Sales Analysis using Power BI

**Overview**
An interactive Power BI dashboard that analyzes Amazon product sales, reviews, and performance metrics.

**Objective**
- Clean and process raw sales data.  
- Build dashboards to visualize **sales trends**, **product performance**, and **customer reviews**.  
- Help identify top-performing products and categories.

**Tools & Technologies**
- **Power BI**
- **Microsoft Excel**
- **Power Query**
- **GitHub**

**Added professional README for project documentation**
1. **Data Cleaning & Processing:** Removed duplicates, corrected data types, and standardized columns using Power Query.  
2. **Data Analysis:** Created calculated columns and measures for KPIs like YTD Sales, QTD Sales, and Reviews.  
3. **Visualization:** Built dashboards showing Sales by Month, Product Category, and Top Products.  
4. **Insights:** Highlighted seasonal trends and top 5 products by YTD Sales and Reviews.

**Dashboard Highlights**
- **KPIs:** YTD Sales, QTD Sales, Total Products Sold, YTD Reviews.  
- **Charts:** Sales by Month, Product Category, and Top Products.  
- **Filters:** Year and Category filters for dynamic analysis.
